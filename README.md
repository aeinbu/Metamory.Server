# Metamory.Server

## Development setup

### Set up ASP.NET Core SSL certificate for development.
At the command prompt, run:
```console
dotnet dev-certs https --trust
```

### To run the server
At the command prompt, run:
```console
dotnet run --project Metamory.WebApi
```
